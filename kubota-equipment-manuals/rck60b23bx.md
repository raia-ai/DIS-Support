## TRACTOR/MOWER PREPARATION

> **CAUTION**
> To avoid personal injury:
> - Park the machine on a firm and level surface.
> - Apply the parking brake.
> - Stop the engine and remove the key.
> - Discharge deflector must remain installed at all times.

## BOOT KIT INSTALLATION

**NOTE:**
<>: Ref. No. of PARTS LIST

### [RCK60B-23BX, RCK54-23BX]

1.  Install the deck stay to the mower deck with the M8 sems bolts and the nuts.
    
    *   (1) Deck stay <5>
    *   (2) M8 x 25 sems bolt <6>
    *   (3) M8 nut <7>

2.  Install the fulcrum stay and mower plate to the mower deck with the M8 SQU. neck bolts, washers, nuts, M10 sems bolt and the nut.

    **RCK54-23BX / RCK60B-23BX**
    *   (1) Fulcrum stay <2>
    *   (2) Mower plate <8>
    *   (3) M8 SQU. neck bolt <9>
    *   (4) M8 nut <7> and washer <10>
    *   (5) M10 x 20 sems bolt <14>
    *   (6) M10 nut <15>

### [RCK48-18BX]

1.  Install the deck stay to the mower deck with the M8 sems bolts and the nuts.
    *   (1) Deck stay <5>
    *   (2) M8 x 25 sems bolt <6>
    *   (3) M8 nut <7>

2.  Install the mower plate to the mower deck with a M8 sems bolt, M8 SQU. neck bolt, washer and a nut. And then install the fulcrum stay to the mower plate with the M8 SQU. neck bolts, washers and nuts.
    *   (1) Fulcrum stay <2>
    *   (2) Mower plate <8>
    *   (3) M8 SQU. neck bolt <9>
    *   (4) M8 nut <7> and washer <10>
    *   (5) M8 x 25 sems bolt <6>

### Installation of the Boot

3.  Install the boot with the pins.

    **Front View**
    *   (1) Boot <1>
    *   (2) Boot pin <4>
    *   (3) Spring pin <3> (Already installed)

    **Rear View <RCK54-23BX, RCK48-18BX>**
    *   (1) Boot <1>
    *   (2) Pin
    *   (3) Deck stay <5>
    *   (4) M8 x 25 sems bolt
    *   (5) Boot stay
    *   (R) "REPOSITION"

    **Rear View <RCK60B-23BX>**
    *   (1) Boot <1>
    *   (2) Pin
    *   (3) Deck stay <5>
    *   (4) M8 x 25 sems bolt
    *   (5) Boot stay
    *   (6) Boot boss
    *   (7) M8 nut
    *   (A) Installing position
    *   (R) "REPOSITION"

> **NOTE:**
> - Make sure the boot stay is installed at the position (A). If not, reinstall the boot stay at the position (A).
> - If the boot is hard to fit, loosen the M8 x 25 sems bolts (4) and reposition the boot stay up and down.

### <Only for RCK60B-23BX>

4.  As shown in the figure, set the boot seal onto the boot inlet.
    *   (1) Boot seal <16>
    *   (2) Boot
    *   (3) Chute 1 <11>
    *   (4) Hose clamp <12>

5.  Install the chute 1 to the boot inlet and blower case inlet with hose clamps. See "Chute installation" in "SETTING UP INSTRUCTIONS" section in your grass catcher operator's manual.
    *   (1) Chute 1 <11>
    *   (2) Hose clamp <12>
    *   (A) Blower case
    *   (B) Blower case inlet
    *   (C) Boot inlet

### Conversion to Side Discharge

6.  To convert the mower deck to the side discharge.
    1.  Remove the pins.
    2.  Remove the boot assembly.
    3.  Remove the mower plate.

> **NOTE:**
> - See "REMOVING THE GRASS CATCHER FOR SIDE DISCHARGE MOWER" in "OPERATION" section in your grass catcher operator's manual.
