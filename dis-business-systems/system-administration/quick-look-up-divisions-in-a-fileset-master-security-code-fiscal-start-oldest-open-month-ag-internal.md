_Quick Look up - Divisions in a Fileset - Master Security code - Fiscal Start - Oldest Open month - Ag Internal_

Issue 

 How to easily find out quick information about divisions in a Fileset 

  

 Resolution 

 DSPPFM MASP or DSPPFM FILEu/MASP  (Where 'u'= the fileset you need information from.) 

  

  

   

  

 Master security division = @ (See position 1 of line 1) 

 Number of active divisions = 02 (See position 7 and 8 of Line 1) 

 Active division are M and N ( see position 9 and 14 of line 1 

 Additional division is noted in each line below.  

  

 To find the Fiscal Start Look at position 150 

 Oldest option month is in position 151 

  

 In the Control Field at the top type W150 to move the columns over to position 150.
