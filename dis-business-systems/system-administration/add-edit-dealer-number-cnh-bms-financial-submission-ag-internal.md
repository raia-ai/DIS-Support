_Add/Edit Dealer Number CNH BMS Financial Submission - AG Internal_

Introduction             

 The CNH dealer portal requires the correct dealer number in order to submit the CNH BMS monthly report. This document details where to add/change the dealer number for the CNH BMS Financial Submission. 

  

 To Add or Edit Dealer Number 

 During the submission process, enter your Dealer Number in the field provided on the Dealer Number screen and click Continue .  

   

 The user will then be presented with the option of printing the report without submitting it, cancelling, or submitting the data to CNH. The system will remember the Dealer Number the next time this process is completed.   

 *** No changes or additions are required in Interface Manager.
