_Can’t Print on Network Printers Using Wifi - AG Internal_

Issue 

 User has no issue connecting to Quantum or Keystone and dealership is able to print wired printers, but cannot print using wifi. 

  

 Solution 

 Update firewall firmware on SonicWall.
