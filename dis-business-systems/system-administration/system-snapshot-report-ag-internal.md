_System SNAPSHOT Report - AG Internal_

Issue 
 Customer or IBM request a report showing all AS/400 system info. 
   
 Solution 
 Open a command line, type SNAP , and click Enter .  The AS/400 System Snapshot report is generated and includes the following information: 
 Operating system level 
 Serial number 
 Disk drives, including type-model and capacity 
 System model number 
 System memory (RAM) 
 Communication resources 
 Scheduled job entries
