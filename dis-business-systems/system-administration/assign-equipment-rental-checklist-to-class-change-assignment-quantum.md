_Assign Equipment Rental Checklist to Class/Change Assignment - Quantum - AG Internal_

Introduction 

 Work with Consignment Checklists allows you to create checklists and assign them to different classes of units. Checklists print on Delivery/Pickup tickets and can be used for a variety of reasons including to ensure that everything that needs to be done when a unit is delivered or picked up has been completed. 

 Checklists must be created before assigning to a class. Click  here  for instructions on creating a checklist. 

  

 To Assign a Checklist to a Class 

 Select  Work with Consignment Checklist (Equipment Rental | Maintenance | Work with Consignment Checklist) .  

 

 Click inside the Unit Class column, type the class to which you are assigning the checklist, and click  Enter . If you do not know the name of the class, you can click the  Select Class  button and select it from the list that appears. 

 Click inside the Checklist Name column, type the name of the checklist that you are assigning to the Unit Class, and click  Enter . If you do not know the name of the checklist that you would like to assign, click the Select Checklist button and select it from the list that appears. 

 The fields of the table are cleared and a message appears at the bottom of the screen stating the additions were successful. 

 Click  Exit  to return to the main menu. 

  

 To Change Checklist Class Assignment 

 Access the Work with Consignment Checklist - Assign Checklist screen, select the checklist that you would like to change, and type the name of the checklist that you would like to assign to the selected class. If you do not know the name of the checklist that you would like to change to, you can highlight the checklist, click the Select Checklist button, and select the new checklist to assign. 

 A message appears at the bottom of the screen stating the Updates were successful. 

  

 To Delete a Checklist Class Assignment 

 Access the Work with Consignment Checklist - Assign Checklist screen, select the checklist assignment that you would like to remove and click the  Delete  button. The assignment is removed from the table and a message appears at the bottom of the screen stating the Checklist assignment of CLASS to LIST has been deleted.
