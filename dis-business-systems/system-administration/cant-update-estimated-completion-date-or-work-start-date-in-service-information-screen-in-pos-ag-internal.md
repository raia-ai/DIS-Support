_Can't Update Estimated Completion Date or Work Start Date in Service Information Screen in P.O.S. - Quantum_

Issue 

 User cannot update the Work Start Date or Estimated Completion Date fields on the Service Information screen in Point of Sale. 

 If a workorder has any of the following information added in Service Scheduling: 

 Any schedule side notes  
 A title  
 Any tag values assigned to the workorder 

 The Estimated Completion Date and Work Start Date will no longer be updateable in Point of Sale. 

  

 Solution 

 Use the Service Scheduling application to update these fields.
