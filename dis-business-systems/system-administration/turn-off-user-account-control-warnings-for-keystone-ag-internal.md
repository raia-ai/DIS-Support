Introduction 

 This article details how to configure windows to prevent the User Account Control Warning from popping up when using Keystone. 

 Configure Computer 

 Press the Windows and R keys at the same time. 

 

 Type Control Panel and click OK . 

 

 Select User Accounts on the screen that displays, then select User Accounts again on the next screen. 

 

 Select Change User Account Control Settings . 

 

 Move the slider to Never Notify and click OK as shown above.
