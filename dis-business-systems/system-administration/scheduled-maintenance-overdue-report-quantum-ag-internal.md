_Scheduled Maintenance Overdue Report - Quantum_

Introduction 

 This is a Fleet Management Report. The Scheduled Maintenance Overdue Report allows you to generate a list of all scheduled maintenance that has not been completed and is past its due date. The report lists the customer information, unit information, estimated hours to complete labor, and cost of labor. 

  

 To Run a Scheduled Maintenance Overdue Report 

 Select  Scheduled Maintenance Overdue (Service | Service Reports | Scheduled Maintenance Overdue) . The Fleet Management Report screen opens. 

 

 Type the ID of the printer that you would like to print the report on and click  OK . 

 A Message appears stating the Scheduled Maintenance Overdue Report has been submitted to print. 

 Click  OK  to return to the Fleet Management Report screen. 

 Click  Exit  to return to the main menu.
