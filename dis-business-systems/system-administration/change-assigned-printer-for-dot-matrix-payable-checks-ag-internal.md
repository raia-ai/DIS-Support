_Change Assigned Printer for Dot Matrix Payable Checks - AG Internal_

Issue 

 This article provides instruction on how to change the assigned printer for customer who prints payable checks on a dot matrix printer.   

  

 Solution 

 The printer for dot matrix payable checks is assigned in Security Maintenance. 

 Select Security Maintenance (Security | Security Maintenance) . 

 Provide the Master security code and click Enter . 

 Provide the Division Code and click Enter . 

   

 Click the Additional Information tab. 

   

 Enter the name for the new dot matrix printer being assigned in the Pay Check Printer field and click Enter to save.
