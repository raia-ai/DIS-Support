_Point of Sale Voided Document Data Mine Subcategory Missing/Doesn’t Provide Results – AG Internal_

Issue 

 Point of Sale Voided Document Data Mine Subcategory does not display or doesn’t provide any results when a query is run. 

  

 Solution 

 Check to see if the u.SAMM5 is there. If there isn’t a u.SAMM5 file, run  Initial File Setup   (Security | Initial File Setup)  X6-7 .
