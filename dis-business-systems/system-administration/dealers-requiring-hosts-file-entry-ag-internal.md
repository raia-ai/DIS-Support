Below is a list from Mike Stanovich with a list of dealers that you must have a HOSTS file entry for to connect to their webclient. 

 Edit the Windows Hosts File 

 The hosts file is located in the etc folder (C:\Windows\system32\drivers\etc). 

 

 You need administrative permissions to edit it, so you'll need to run Notepad as administrator to edit the HOSTs file. Click the Windows Start button, type Notepad into the search bar, right-click Notepad , and click Run as administrator .
