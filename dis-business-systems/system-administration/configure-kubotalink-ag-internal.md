Introduction 

 This article details how to use KubotaLink. 

 KubotaLink is an online feature available on Kubota’s official websites that allows authorized dealers and other users to view DIS customer locations for the purpose of placing parts orders as shown below. 

 

 To Configure KubotaLink : 

 KubotaLink needs to be configured in each Division. 

 Select Vendor Maintenance – Parts Locator (Manufacturers | Kubota | Vendor Maintenance – Parts Locator). 

 

 The dealer’s vendor code for Kubota is used and there is no need to add Part Class. Dealer Code lookup is found in ECM. The information is uploaded during scheduled nightly data processing .
