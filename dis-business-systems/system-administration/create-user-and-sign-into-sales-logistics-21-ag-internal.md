_Create User and Sign into Sales Logistics 2.1 - AG Internal_

Issue 

 User Sign In was changed in Sales Logistics Release 2.1 

 

 You are no longer required to Generate Sales Logistics Code in Prism and the associated button in the screen shot below has been removed. 

 

  

 Solution 

 Users are still created in Prism Admin. 

 

 Click the Prism icon. 

 

 Use the following sign on information: 

 User: dis@ dealernumber .com 
 Password: puget 

 Click Log In . 

 

 Click the Administration (gear) icon in the upper left, select Staff and click Add . 

 Select the Enable Sales Logistics check box. 

 That’s all the set up that’s required in Prism. 

  

 Download Sales Logistics from the App store. 

 When signing in for the first time enter the following: 

 Enterprise ID:  Dealer Number 
 Username:  Email address set up in Prism Admin 
 Password: Password set up in Prism Admin.
