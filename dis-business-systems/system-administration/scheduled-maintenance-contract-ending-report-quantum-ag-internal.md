_Scheduled Maintenance Contract Ending Report - Quantum_

Introduction 

 This report allows you to generate a list of all scheduled maintenance contracts that are set to end in a specified month. This report provides the following information: 

 Total number of units with contracts set to end 
 Addresses of customers that own the units 
 Exact date contract ends 
 Hourly rate charged for each maintenance contract 
 Unit model and serial number 
 Maintenance frequency 

  

 To Run a Scheduled Maintenance Contract Ending Report 

 Select  Scheduled Maintenance Contract Ending (Service | Service Reports | Scheduled Maintenance Contract Ending) . The Fleet Management Report - Contracts Ending screen opens. 

   

 Type a month and year in the field provided using the MMYY format or click the calendar icon to select the month/year from the calendar that appears. All scheduled maintenance contracts that end in the month provided will be listed on the report. 

 Type the ID for the printer that you would like to use to print the report. 

 Click  OK . A message appears at the bottom of the screen stating that the report has been submitted to print. 

 Click  Exit  to return to the main menu.
