_Ensure readwrite permissions are granted to user_

The error in the red box below is related to a lack of permissions. 

 

  

 To Resolve Error 

  

 

 Select the user name from the Database connection in the left pane (Server Objects | roles | username). 

 

 Confirm that the readwrite permissions are granted to the user and add them if they are not. 

 Note : Reach out to Chris Latham with the above details if you cannot add the necessary permissions.
