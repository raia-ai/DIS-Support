## INTRODUCTION

The DIS Interface Manager application provides ECC – Electronic Commerce Client and EPC – Electronic Parts Catalog interfaces for a variety of manufacturers. Most dealerships use some part of Interface Manager. Although Interface Manager is easy to use, online help for interface manager is available.

The Interface Manager On-line Documentation breaks down aspects of Interface Manager including how to configure and use Electronic Commerce Client (ECC) and Electronic Parts Catalogs (EPC). It also provides information on configuring and using Shiplink and Barcoding. Additionally, you can connect to LogMeIn from the Help drop-down menu.
