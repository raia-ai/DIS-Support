## ACCOUNTING

### New Screen Layout and Features for Create Source Documents (FOR KEYSTONE ONLY)
*(Accounting | Create Source Documents)*

We've changed the screen layout and added new features to the Create Source Document - Document Entry screen. The new features that have been added have red numbers by them in the screenshot and are described in the correlating points on the pages that follow.

The screenshot of the "POST SOURCE DOCUMENTS - Document Entry" screen highlights the following new features, identified by red numbers:
1.  **Debit/Credit/Difference Box**: Located at the top, showing debit, credit, and difference totals.
2.  **Change Button**: A new button for editing existing lines.
3.  **Delete Button**: A new button for deleting existing lines.
4.  **Line# Column**: A new column in the transaction grid showing the line number.
5.  **Line# Field**: An input field at the bottom for entering a new line number.
6.  **Account# Look Up**: The "Account#" column header is now a clickable link for lookups.
7.  **Date Look Up**: The "Date" column header is now a clickable link. (Note: Text describes Address# and Stock# lookups, not Date).
8.  **Stock# Look Up**: The "Stock#" column header is now a clickable link for lookups.
