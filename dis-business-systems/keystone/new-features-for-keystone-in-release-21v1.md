## INTRODUCTION
The following information is on the primary new features in R21v1. Complete step-by-step instructions for all new features will be added to Keystone-Online-Help, as soon as they are available.
If you are interested in purchasing Keystone or any of the other products, please call DIS at 1-800-426-8870.

## UNITS
The following changes involving the Units menu option have been made:

### Make/Model and Customer Unit Number Length Extended
- The Maximum Make name length has been extended to 10 characters and the maximum Model name length to 16 characters.
- The maximum Customer Unit number length has been extended from 6 to 17 characters in the options under the P.O.S. and Service Menus.

### The Number of Unit Pictures that Can be Added Has Increased to 999
- The number of unit pictures that can be added and stored for each unit has increased from 6 to 999 (must use Windows 7 and above - Not XP).
- A general comment can be added on the Unit Pictures entry page and a specific comment can be added for each individual picture of the unit.
- Viewing and uploading the pictures is now faster.
- Pictures that have been saved can be accessed in Sales Logistics.
- [Click here](LINK) and run the Unit Pictures update.

> **Note:** Only images in Unit Pictures in Inquire/Update Units can be accessed in Sales Logistics. If you have added pictures in EasyFile rather than Unit Pictures, the .jpg files can be transferred. Call DIS for assistance.

### Units Defined Search Options and Results Enhanced
- You can now search for primary units configured with an attachment, units designated as attachments, and/or units that do not have attachments when searching for Units in Define Search.
- Search results now include units from all divisions by default.

### Unit Details Can Be Configured to Display in Order of Choice on Inquire/Update - Unit Selection Screen
- Click the Configure Unit List button to customize your view.
