## Introduction

### Price File Interface
The new Price File interface for Polaris Industries provides daily price list updates for the USA and Canada. The business system will search for price updates twice daily and complete the following if updates exist:
- Update the prices in the price book assigned to the Polaris Vendor Code
- Record the number of parts that had a price change
- Notify a specified user of the number of parts affected by the price change

### Unit Promotions Interface
The new Units Promotion Interface allows you to look up active promotions for specified Polaris models. Promotion options include rebates, extended service contracts, Financing, and SPIFFs.
