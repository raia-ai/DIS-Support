Introduction 

 This article details how to add a new user to Altura and then enable them as a Salesperson in Sales Logistics. 

 Â 

 Add Altura User 

 If the user has already been added as an Altura user, skip to the Enable Sales Logistics User section. 

 Access Altura. 

 

 Select User Management (Admin Settings | Staff Users | User Management) . The User Management screen displays. 
 Select the ALL USERS tab if it is not already selected. 
 Select the + Add User button. The New Staff User screen displays. 
 Provide the new user information and click the Submit button. The user is added to Altura. 

 Enable Salesperson in Sales Logistics 

 The user must be added to Altura before they can be added as a Salesperson. 

 Access the User Management screen as detailed in the previous section. 

 

 Select the SALESPERSON MANAGEMENT tab. If the tab is not displayed, contact DIS Support for assistance. 
 Select the + Add Salesperson button that appears in the upper right corner. 
 Click on the Salesperson field, select the user that to add as a Salesperson. 
 By default, the Show Cost check box is selected. Leave it selected to allow the user to view the cost pricing of units in Sales Logistics. Unselect the check box to hide cost pricing. 
 Click Save .
