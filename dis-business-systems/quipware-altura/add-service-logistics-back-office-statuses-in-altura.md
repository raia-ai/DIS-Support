Introduction 

 Important! The process in this document is only applicable to Quipware, Solutions, and MiniTrac dealerships. Quantum dealerships can use Reason Codes  to create similar statuses.  

 This document details the configuration that needs to be completed in Altura for a user to create new Back Office Statuses for Service Logistics. 

 Enable Permissions 

 The Admin Settings for Configurations > Service option must be enabled for users to be able to Add and Update Back Office Statuses.  

 Select Admin Settings (Roles and Permissions | Admin Settings) . 

 

 Enable the Configurations > Setting permission to give the user access to all Service related configurations. 

 Add, Update, Delete Back Office Statuses 

 Once the permissions have been enabled, the new Back Office statuses can be added and updated. 

 

 Select Service (Admin Settings | Configurations | Service) . The Configurations Service screen displays. 

 

 Select the Back Office Statuses option in the upper-left corner of the screen. 
 Click the + ADD button. The Add Back Office Status screen displays. 

 

 Enter the name of the new Back Office Status in the field provided. 
 Click Save . The New Back Office Status displays in the table. 

 

 Enable the new Status by sliding the switch. Be sure it is colored as shown above. 
 Click the ellipsis on the Back Office Status line to update the Back Office Status name or delete it entirely. If you select to update the status name, the Edit Back Office Status Screen displays. 

 

 Click on the status name and update it as desired. 
 Click Save to update the Back Office Status name.
