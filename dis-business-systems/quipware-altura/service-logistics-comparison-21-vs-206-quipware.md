_Service Logistics Comparison - 2.1 vs 2.0.6 (Clone)_

Changes to the Overview screen: 

 Color coding on the buttons for Overview, Diagnostics, Checklist, Repair Work, and Review 
 Green = Ok 
 Yellowing = Warning 
 Red = Critical 

 Inventory and Add buttons removed from the top. A + button has been added below the banner to add a segment or quote. 
 A Sync button has been added to sync changes to this specific WO on demand. 
 Segment Title and Customer Complaint have been moved to the top of the frame. 
 Technician Info at the bottom of the screen shows all techs currently assigned to the WO on the left and Previously assigned techs on the right. 
 Address validation has been added to the Globe button giving the tech the ability to edit the address. 

  

 

  

  

 In 2.1 Failure Code has been moved to Step 2 of the Job Review. 

  

 

  

  

 2.1 allows for Push notifications on the I Pad. 

  

 

  

 2.1 Improved workflow in Timesheet and added the ability to close out blank days as a block. 

  

 

  

  

 2.0.6 Workorder search excluded completed Workorders. In 2.1 a new button was added to view them. 

  

 

  

 2.1 Added the ability to attach photos to quotes and now allows the tech to access the photo roll.
