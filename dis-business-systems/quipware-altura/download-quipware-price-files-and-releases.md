Introduction 

 This article details how to download QuipWare Price Files and Releases using FileZilla which DIS recommends for FTP transfers. 

 Download and Install FileZilla 

 The FileZilla segment of this document was created on 12.1.25. FileZilla may have changed this process since then. 

 Select the link below and choose the appropriate file from those listed. Modern versions of Windows would choose the 64bit setup file. 

 https://filezilla-project.org/download.php?show_all=1 

 Access the Downloads folder on your PC. Once the installer has been downloaded, you can find it in your Downloads folder. 

 Double click the .exe file to start the install process. 
 Click the link below for installation instructions 

 https://wiki.filezilla-project.org/Client_Installation 

  

 Setup a Connection to ftp.quipware.com 

 Open FileZilla. 

 

 Select Site Manager (File | Site Manager). 

 

 Click the New Site button. 

 

 Provide the following information in the listed fields: 

 Protocol – Click the drop-down and select the FTP – File Transfer Protocol option. 
 Host – Provide a name for the connection that makes the most sense to your dealership. It has been named ftp.quipware.com in the example above. 
 Port – 21 
 Encryption – Click the drop-down and select Use explicit FTP over TLS if available . 
 Logon Type – Click the drop-down and select Ask for password . 

 Note : When you select Ask for password, the following screen displays. Some devices may allow you to set this to Normal and save the password. 

 

 User – provide a personalized FTP UserName – this does NOT need to be enclosed in carats as shown in the image above. 

 Click OK to save the connection information. 

 

 From the FileZilla home screen, click the drop-down next to the Site Manager button and select the newly created connection. The Enter Password screen displays. 

 

 Provide the Password and click OK to connect. The following screen displays when connected, listing the files stored on your computer on the left in the Local Site section and the files available for download on the right side in the Remote Site section. 

 

 Click on Folders to reveal additional Folders and Files. 
 Use the available options to download desired files to your system. Price files and QuipWare releases are located in their respective folders for your use.
