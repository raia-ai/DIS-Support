_Export Cycle Count to Altura and Log Count List Information – Quantum / Altura_

Introduction 

 This article details the process of performing a cycle count with Quantum and Altura including how to: 

 Manually generate a cycle count in Quantum and export it to Altura 
 Access the cycle count in Altura from a tablet, phone, or PC 
 Print the exported cycle count list 
 Log counted part quantities, assign a single bin location to a part, add parts to the system that are not currently listed in bins / the system (Vendor must already be in system) 
 Submit logged cycle count information back to the DMS and view in Quantum 

 Note: Click the link at the end of this article to view the PDF of this article.

  

 Create Manual Cycle Count 

 Select Cycle Count Process (Export and Validation) (Parts | Parts Inventory Adjustment | Parts Inventory Cycle Count | Cycle Count Process (Export and Validation)). The Cycle Count Process screen displays. 

 

 Select the Create Manual Count button. The Create Manual Cycle Count screen displays. 

 

 Use the associated check boxes and / or available fields to create the desired cycle count list. 

 Select Continue . 

 Select Yes on the submit verification screen that displays. A message displays at the bottom of the screen stating the Parts count list creation has been submitted . 

 

 Select the Export Count List button. The Export screen displays. 

 

 Highlight the count list to be exported and choose the Select button. The next export screen displays. 

 

 Important! You MUST enter the letter B followed by a number in the scanner number as shown above to export the count list to Altura. If the letter B is not entered, the count list will be sent to the associated scanner instead of Altura. 

 Select Continue . The count list is exported. 

   

 Access Count List in Altura and Provide Count Information 

 Access Altura. 

 

 Select Parts , then select the Cycle Count option. The Cycle Count List screen displays. 

 Select the cycle count list that you would like to process. The cycle count List displays. 

 

 The name assigned to the count list in Quantum is listed at the top of the screen in addition to the date the count was generated. 

 Next, the total number of parts that have been counted out of all of the parts included in the list as well as the current Status assigned to the list is displayed. The count total is updated after information is added and applied for each part. 

 After the count total, the name of the division/branch that generated the count list, total number of parts included in the count list, and the date the count list is generated is provided. 

 In the table, all of the parts and their associated information are listed on a line-by-line basis. 

 The following options can be completed from this screen and are detailed in the sections that follow: 

 Print the count list 
 Configure the parts included in the list and search for specific parts 
 Update inventory quantity/bin location 
 Add Parts found while taking inventory that are not currently on the list 
 Upload the count information to the DMS (Document Management System)/Quantum 

   

 Print Count List 

 A version of the Cycle Count List with space for entering count numbers and bin locations can be printed if desired. This option also allows you to add a header and footer as well as shading to the list. 

 

 Select the Print button while on the Cycle Count screen. The Print screen for the browser that you are on opens (Chrome is being used in this example). 

 Important!   You may need to remove the popup blocker for the browser you are using if the Print options do not display. 

 

 If you would like to include the date and time the count list was printed, the list ID number, and page numbers, choose the More settings drop-down arrow then select the Headers and footers check box. 
 If you would like to utilize available shading options, choose the More settings drop-down arrow then select the Background graphics check box. 

 When finished configuring print options, select the Print button to generate the list. 

  

 Configure Part List Information / Search for Parts on List 

 The count list can be filtered and sorted so only the parts that you want are included in the list. Parts included on the list can also be located using the Search option. 

 

  

 Work with Filters 

 Select the Filters button to set a filter(s), select a saved filter, or clear set filter(s). 

 

 To create and apply a new filter, select the MFG drop-down arrow and specify which vendor(s) parts that you want to include in the list AND/OR select the Status drop-down arrow and select whether you want to display COUNTED parts, UNCOUNTED parts, or ALL parts. 

 Select Apply to enable and set the filter. If you want to use this filter at a later time, select the Save Filter button, assign it a name, and specify whether everyone one at your dealership can use it (Public) or if it can only be used/seen by yourself (Mine). 

 To apply a saved filter, select the filter from those saved in the Saved Filters list on the left side of the screen. It is automatically applied. 
 To clear a set Filter, select the Clear button. 

  

 Sort Parts in List by Column 

 Select the Sort button to order the parts in the count list according to the information in a specific column. 

 

 Locate the column header that you want to organize information by, then select whether you want the information to display in ascending or descending order. In the example above the parts in the Count List are ordered in descending order according to the Part Description. 

  

 Locate Part(s) in Count List 

 You can locate a part or parts in the Count List by entering any portion of its Part Item (number), Bin Location, or Description in the Search field and selecting the Magnifying Glass . 

 

 In the example above, TRA has been entered in the search field. Selecting the Magnifying Glass narrows the parts in the Count list to just those with TRA in one of the fields mentioned above. 

  

 Update Inventory Quantity / Bin Location 

 Once you are ready to enter counted inventory and/or bin location information, access the count list using the instructions in this document if it is not already open. 

 

 Tap or click in the Qty Counted field on the same line as the part being updated and type the quantity counted or select the arrows in that field until you reach the correct quantity. 

 Update or enter the bin location for the part. 

 Select the Apply button that displayed when you selected one of the fields on the part line. The part line turns green, and a message is displayed at the bottom of the screen stating Cycle Count List Line updated successfully as shown below. 

 

 Continue entering/updating the quantity and/or bin location information for each part that was counted. Select the Apply button foreach Count List Line when count information has been provided. 

  

 Add Part not on Count List 

 If you come across a part during the inventory process that is not on the Count List, you can add it and assign a single bin location if desired. 

 

 Select the + Add Part button in the top-right corner of the screen. The Add Part screen displays. 

 

 Provide the part information in the provided fields and select Save when finished. 

 Important! The part manufacturer must already be in the system in order to add their part to your inventory. 

 If the part is successfully added to your inventory, the Add Part screen closes, and a message appears at the bottom of the screen stating Part added successfully as shown below. 

 

  

 Upload Count Information to DMS 

 Once you have entered all of the count information and added any desired parts, the information needs to be uploaded to the DMS. 

 

 Select the Upload to DMS button in the upper-right corner of the screen. After a moment, if the upload is successful, a message displays at the bottom of the screen stating the Cycle count sent to DMS as shown below. 

 

 The information is updated in the DMS/Quantum.
