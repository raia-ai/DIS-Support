_AP Payment Third Party Export_

Third Party AP Payment Export 
 QuipWare Version 4.6.20.0 
 Define payment as 3PRTY and specify account counter to use. Open the Supplier module and select Payment Method under Setup. Create a new or update existing payment. 
 
 Setup output type for 3PRTY payment, Open GL Module and select Tools, Third Party Payment. 

 Select the Output Type and Output Location and Save the settings. 
 
 
 Run the AP Payment process as usual, any payment types for 3PRTY payment will get picked up If 3PRTY payments are picked up you will receive two dialogs during the process.
